## Getting Started

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file in the root directory and add API endpoint:
   ```bash
    cp .env.example .env
   ```

3. Checkout https://github.com/jinglechen2287/si679-final-server.git for the API Code

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Follow https://vercel.com/docs/git#deploying-a-git-repository to deploy to vercel
